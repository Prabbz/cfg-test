Si vous essayez d’utiliser Safari alors que vous êtes hors ligne

Vous devez connecter votre Mac à Internet pour accéder à une page web dans Safari sauf si vous avez ajouté cette page dans votre liste de lecture lorsque vous étiez en ligne.

  • Il se peut que votre Mac soit temporairement déconnecté d’Internet si vous l’avez déplacé dans un endroit non équipé d’un réseau Wi-Fi, ou ce dernier ne fonctionne pas. Réessayez lorsque votre Mac se trouve dans un endroit doté d’un réseau Wi-Fi fonctionnel.

  • Il se peut que votre connexion Internet soit hors service en raison d’un routeur ou d’un modem défaillant, ou d’une panne du côté de votre fournisseur d’accès Internet (FAI). Pour résoudre les problèmes liés à la connexion Internet, utilisez la fonction Diagnostic réseau.

    Ouvrir Diagnostic réseau

  • Lorsque vous êtes hors ligne, vous pouvez lire des articles déjà ajoutés à votre liste de lecture. Pour l’afficher, cliquez sur le bouton Barre latérale dans la barre d’outils, puis sur le bouton Liste de lecture en haut de la barre latérale.
